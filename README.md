# getmicah.github.io
My blog made with [jekyll](http://jekyllrb.com) and hosted with Github Pages.


### Install

* fork this repository
* edit the `_config.yml` with your info
* change the links in `_data/navigation.yml`
* remove my posts from `_posts/`
* rename your repo to ***your-username*.github.io**

### Local jekyll view

* git clone `https://github.com/ikivil/ikivil.github.io.git`
* 进入目录 `cd ikivil.github.io`
* jekyll serve
* 浏览器输入地址 `http://localhost:4000/`
